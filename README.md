#### 系统采用技术:  
> 本系统采用  JDK1.8 + MyEclipse开发工具 + MySQL5.6版本数据库 + Tomcat服务器 8.0.33  
>
> 后台：  
> SSH 框架  ：Struts 2.3.20 + Spring 3.0.2 + Hibernate 3  
> Apache 相关组件：logging 1.1.1 + log4j1.2.15 + fileupload1.3.1 等。。。  

> 前端：  
> 前端框架：Amaze UI  
> 富文本编辑器：百度ueditor组件  
> MarkDown组件：Editor.md
